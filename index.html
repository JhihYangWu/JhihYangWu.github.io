minigrad
This project all started when I was lying in bed watching George Hotz make a PyTorch clone from scratch called tinygrad. I was so amazed that someone even has the courage to try to make a deep learning framework from scratch. So after watching his stream recording for a few days and reading hundreds of commit history, I decided I want to do the same, and that is how minigrad was born. Most people have no idea how the autograd engine works in modern deep learning frameworks, they just use it. Building minigrad forced me to truly understand how deep learning works under the hood and how creating a computation graph and using topological sort you could compute all the gradients in an efficient way. I also had to implement all the operations and optimizers from scratch which forced me to have a very good understanding of things like softmax, conv2d, batchnorm, Adam, etc. Of course I had to use my own library to train a few neural networks so I trained a CNN to classify MNIST digits, a LSTM to classify which language a last name is in, and an Attention model to translate dates into standardized format.

armCPUonFPGA
With the recent boom in generative AI and hardware acceleration, I became really interested in how chips are built. In armCPUonFPGA, I implement a 32-bit single-cycle ARMv4 CPU using the hardware description language Verilog and use it to program my FPGA (Tul Pynq-Z2). I wanted to understand how each component (carry lookahead adder, ALU, multiplier, MUX, etc) works in real life so I limited myself to only using logic gates and building up from there. I would love to turn my code into an actual chip and get it manufactured at a fab such as TSMC but since I don't have the money or knowledge, the only way to turn my code into hardware is using FPGA. This project combines the knowledge I learned in my computer architecture class and digital logic class while also giving me the chance to learn a more popular computer architecture. Check out my GitHub for more information about this project.

(This project has been outmatched by MIPS CPU on FPGA created while I was taking ECE 369A at UofA. The repo is not public because it is coursework.)

Everything Controller
It's about time that I do a hardware project so I thought how cool would it be to make a PS4 controller clone that could be used to play games on-device, act as a wireless controller on PC, and be a RC transmitter for drones. At the heart of Everything Controller is an ESP32 which reads input from 10 buttons and 2 joysticks. It also controls the 3.2 inch TFT touch screen display as well as the NRF24L01 wireless transceiver. This is all powered by two 1.5 V batteries and a boost converter but I found connecting it to a usb port/charger using a wire is more reliable. All of this is connected using two custom designed PCBs that I got manufactured by JLCPCB. All the components then fit perfectly into my custom designed 3D printed case made using Fusion 360.

Arizona Autonomous
After coming to the University of Arizona for a few weeks, I joined a club called Arizona Autonomous which builds and programs UAVs (unmanned aerial vehicles) to compete in the annual SUAS Competition. I joined the Computer Vision Team and was tasked to make software that could detect and classify letters/shapes on the ground from photos taken by a camera mounted under our UAV. We decided to go with a machine learning approach so I created a Unity project to simulate images we expected to see at the competition. I then implemented the YOLO: Real-Time Object Detection algorithm from scratch in Keras and trained it to draw bounding boxes around the letters/shapes given Canny edge detection filtered images. I then trained a ton of convolution neural networks to classify the attributes of the objects inside those bounding boxes. Since multiple teams compete in the sky at the same time, we needed to create an object avoidance system. To do this, I created a script to generate a 3D vector space around our drone in real time using stereo vision and MiDaS.

Crypto Trading Bot
From April 2022 to August 2022, I developed a trading bot that makes trades between BTC and USDT on the Binance Exchange with hopes of making profits. Instead of using existing trading algorithms such as the Grid Trading Strategy, I invented my own using neural networks and a custom loss function. After training my algorithm on 8 months of Bitcoin price data, I backtested my algorithm on 30 days of unseen data and got 44.39% profit when the price of Bitcoin only went up by 14.64%. Unfortunately, my algorithm did not perform well after those 30 days so I was not able to become rich. I hope to continue working on this project when I have more time.

BookTank
In my junior year of high school at the SMIC Private School in Shanghai, I worked with 2 of my classmates and built a full blown WeChat Mini Program for students and parents to buy and sell school textbooks more effectively. In the mini program, you could see what others have listed for sale and contact the seller with a push of a button. You could also upload pictures of your own books for sale on the platform. We even built a messaging system inside our mini program using WebSockets. As of September 2022, we have attracted more than 1900 users and have more than 1100 book listings of which more than 600 are successful transactions. One of the things that I am most proud of about this project is saving students who needed a textbook really urgenty for class but couldn’t find it anywhere else.
